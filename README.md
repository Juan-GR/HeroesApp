# HeroesApp

A simple app to manage your favorite heroes.

## Features

- List of heroes
- Add a new hero
- Search a hero
- Delete a hero
- Update a hero
- View a hero

## Usage

- Clone the repository
- Run `npm install`
- Up `json-server data/db.json` with `npm run backend`
- Run `ng serve`
- Navigate to http://localhost:4200
